# i-There

</div>

i-There는 아기돼지 삼형제의 집짓기 이야기를 모티프로 한 ADHD 아동을 위한 AR 게임입니다. 아이들은 귀여운 아기 돼지 캐릭터와 함께 재료를 모으고, 오븐에서 벽돌이 구워지는 걸 지켜보고, 직접 벽돌을 쌓아 집을 완성하는 모험을 떠납니다. 파스텔톤의 동화적 공간, 둥글고 친근한 캐릭터, 유기적이고 직관적인 인터랙션, 미션 완료 시 펑펑 터지는 3D 이펙트 보상까지—아이들이 '검사받는다'는 느낌 없이 신나게 놀 수 있도록 설계했습니다. 그 사이 AI가 아이의 시선, 움직임, 음성 패턴을 조용히 분석해 ADHD 증상을 스크리닝하고, 이후 증상의 정도에 따라 행동 발달 훈련을 위해 설계된 콘텐츠의 개인별 맞춤 난이도를 조정해줍니다.
